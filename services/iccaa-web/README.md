# ICCAA Web

This service powers `iccaa.shasvata.com` inside the Shasvata platform monorepo.

## Local development

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open:
   `http://localhost:3003`

## Production

The app is built with Vite and shipped as a static bundle behind Nginx in Docker.
