# Foundation Data Dictionary

## Entity

A company or organization that can have sources, metrics, scores, insights, and reports.

## Source Document

A document or structured source that supports metric extraction.

## Source Citation

A locator and excerpt inside a source document.

## Metric Definition

The canonical definition for a measurable sustainability value.

## Metric Value

A source-backed metric value for an entity and reporting period.

## Score Run

A specific scoring execution tied to a methodology version.

## Score

A methodology-versioned computed value for an entity.
