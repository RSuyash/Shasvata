# App Boundaries

## www

Public site for `shasvata.com`. It owns general public navigation for Intelligence, Advisory, Connect, Impact, Academy, About, Contact, Privacy, and Terms.

## insights

Static content-first Academy and research surface for `insights.shasvata.com`.

## intelligence

Fixture-backed product surface for `intelligence.shasvata.com`. It owns company, sector, compare, reports, methodology, data, and API docs route shells.

## app

Mock authenticated workspace for `app.shasvata.com`. Real auth is deferred.

## admin

Mock operator console for `admin.shasvata.com`. Secure auth must be implemented before production use.
